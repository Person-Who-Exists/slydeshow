# Slydeshow
A (in the works) system of triggers and elements that allows you to create interactive and beautifully animated slideshows

Slydeshow is made of 2 parts slydeshow itself which is a web renderer for slydeshows and slydeshow designer which is a gui app for making slydeshows
## Authors

- [Person who exists](https://gitlab.com/PersonWhoExists) (Main dev)


## Installation
All you need to view a slydeshow is a web browser. If you received a .slyde file you can view it at [slydeshow.dev](https://slydeshow.dev/view). If you received a .slyde.html file you can double click it and it will open right up in your browser

To make a slydeshow you can download a .jar file or .kexe file from the releases tab of the slydeshow designer github repo

```bash
# For .jar
java -jar path/to/jar
# For .kexe
./path/to/.kexe # Mac and Linux
.\path\to\.kexe # Windows
```
